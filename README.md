# Ayan Dhanda - AI Product Manager Portfolio

A professional portfolio website showcasing AI product management case studies.

## Featured Projects

- **Ask AI**: Natural language data query interface (95% time reduction)
- **RAG QnA Chatbot**: Developer Q&A chatbot ($0/month, 70% reduction in clarifications)

## Tech Stack

- Next.js 16 (React)
- TypeScript
- Tailwind CSS v4
- Deployed on Vercel

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Build

```bash
npm run build
npm start
```
